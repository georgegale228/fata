« Savoir faire, pouvoir faire » — Tagline officielle fata

1. Introduction

Ce guide définit le système de points ("kudos") pour reconnaître et récompenser les contributions des ambassadeurs fata.school. Notre mission est d'accélérer l'apprentissage des technologies numériques pour la jeunesse africaine. Chaque action qui avance cette mission mérite d'être valorisée.

Les points sont attribués selon l'impact de chaque contribution sur notre communauté et notre mission éducative.

8. Notes Importantes

  • Attribution : Les points sont attribués par les ambassadeurs lead de chaque pays (Favour pour le Togo, Imane pour la Côte d'Ivoire, Richard ou Joel pour la Guinée), ou les membres du conseil d'administration (Oumie, Djoumé)
  • Actions non listées : Les points peuvent être attribués pour des actions qui ne sont pas listées dans ce document. Chaque action qui avance suffisamment notre mission mérite d'être valorisée.
  • Cumul : Les points sont cumulables pour plusieurs actions dans la même période.
  • Documentation : Les actions doivent être documentées (photos, liens, témoignages) pour être validées.
  • Qualité : La qualité de la contribution peut influencer le nombre de points attribués.
  • Bonus : Des points bonus peuvent être attribués pour des contributions exceptionnelles.
  • Mise à jour : Ce guide peut être mis à jour pour refléter l'évolution des besoins de la communauté.

Procédure d'Attribution

Comment attribuer des points dans l'application fata
1
Aller sur son profil dans l'application fata
2
Cliquer sur Ajouter des ami.e.s
3
Taper le username de l'ambassadeur à qui on veut attribuer des points puis cliquer sur son image
4
Cliquer sur kudos ambassadeur dans la section "Bulletin"
5
Choisir le nombre de points (1, 2, 3 ou 5), ajouter un commentaire, puis cliquer sur Envoyer les kudos
ℹ️ Information

Seuls les Ambassadeurs Lead ont accès à cette fonctionnalité d'attribution de points.

Pour toute question, contactez-nous à info@fata.school
